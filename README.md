# Infengine
An information aggregator that you can use to query AI with, and uncover trends, patterns, and hidden information that may not be obvious  
when viewed separately.

This is meant to run periodically throughout the day, grap new reports, blogs, news feeds, etc, combine them all, create relationships, and to create a  
daily report on what's going on, what might be happening behind the scenes, and some things to look out for in the future.
